# Tekh-bot

Tekh-bot is a technical chatbot built for students and professionals who want fast, practical support in technology.

It helps users:
- Resolve tech-related questions with clear and actionable answers
- Understand concepts, tools, and workflows across modern software topics
- Get career-focused advice to build direction, confidence, and awareness in the tech world

## Product Vision

Tekh-bot is designed to improve tech awareness and decision-making through guided conversation.

The goal is not only to answer questions, but also to support growth by helping users:
- Learn continuously
- Solve real development problems
- Make better career choices in technology

## Core Capabilities

- Conversational technical guidance for day-to-day learning and problem solving
- Context-aware responses using retrieval + web-backed lookup when needed
- Reliable fallback behavior to keep answers available even when a provider is unavailable
- Clean and responsive chat experience across desktop and mobile

## Who It Is For

- Students exploring computer science and software development
- Early-career developers building practical skills
- Working professionals seeking quick technical clarification and career direction

## High-Level Architecture

- `client/`: Web chat interface
- `server/`: API layer with retrieval, web search, and LLM orchestration
